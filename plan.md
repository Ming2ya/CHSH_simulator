# 双比特纠缠与 CHSH 蒙特卡洛验证项目计划

## 摘要

项目按三层推进：先完成可交付 MVP，再做高分稳妥增强，最后视时间冲刺展示型拓展。主线采用 **NumPy 自建量子模拟器**，保证原理透明；同时保留 **Qiskit/Aer 交叉验证**作为提高项。报告使用 **Markdown + 图片**，后续可转 PDF。

核心目标是制备 Bell 态，例如：

```text
|Phi+> = (|00> + |11>) / sqrt(2)
```

在不同测量角度下进行大量 shots 抽样，统计关联期望值：

```text
E(a, b) = P_same - P_diff
```

最终计算 CHSH：

```text
S = E(a, b) + E(a, b') + E(a', b) - E(a', b')
```

展示其超过经典极限 `2`，并随 shots 增加接近量子理论最大值 `2sqrt(2)`。

## MVP 必须实现

- 建立双比特态向量模拟：
  - 初始化 `|00>`
  - 施加 `H` 和 `CNOT`
  - 得到 Bell 态 `|Phi+>`

- 实现任意角度测量：
  - 使用二维实平面测量基，角度参数为 `theta`
  - 对两个量子比特分别选择测量角 `a, a', b, b'`
  - 推荐最优角度：
    - `a = 0`
    - `a' = pi/2`
    - `b = pi/4`
    - `b' = -pi/4`

- 实现 Monte Carlo shots：
  - 对每组角度重复测量 `shots` 次
  - 输出四种结果计数：`++`, `+-`, `-+`, `--`
  - 计算每组关联期望值 `E`
  - 计算 CHSH 值 `S`

- 提供清晰命令行入口：
  - `python main.py --shots 1000`
  - `python main.py --shots 10000 --seed 42`
  - 输出理论值、模拟值、误差、是否违背经典极限

- 保存基础结果：
  - `results/chsh_summary.csv`
  - `figures/chsh_convergence.png`
  - `report.md`

## 稳妥高分功能

- Shots 收敛实验：
  - 比较 `100, 500, 1000, 5000, 10000, 50000` shots
  - 展示 `S` 如何从随机波动逐渐接近 `2sqrt(2)`
  - 图中标出经典极限 `2` 和量子上界 `2sqrt(2)`

- 角度扫描：
  - 固定 Alice/Bob 的一部分角度，扫描另一组角度
  - 画出 `S(theta)` 曲线
  - 展示只有合适测量基才能明显违背 CHSH

- 理论值对照：
  - 对 Bell 态 `|Phi+>` 使用理论公式：
    - `E(theta_a, theta_b) = cos(theta_a - theta_b)`
  - 报告中比较理论期望值、Monte Carlo 估计值和误差

- Qiskit/Aer 交叉验证：
  - 若环境未安装 Qiskit，实施阶段必须先尝试安装
  - 用 Qiskit 构造同样的 Bell 电路
  - 对相同 shots 和角度设置做验证
  - 报告中说明 NumPy 模拟器与 Qiskit 结果一致

- 可复现实验：
  - 所有随机实验支持 `--seed`
  - 所有图和表由脚本生成
  - 报告中的数值来自保存的实验结果，不手抄
